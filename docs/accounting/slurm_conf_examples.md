# Slurm Configuration Examples

Slurm 설정 템플릿 설명

---

# TIME Format

Slurm Time Format

```
HH:MM:SS
D-HH:MM:SS
```

예시

```
00:30:00
1-00:00:00
5-12:30:00
```

---

# NodeList Format

Hostlist 예시

```
compute-[1-4]
gpu-[01-08]
node01
```

---

# Memory Unit

Slurm Memory Unit

단위

```
MB
```

예시

```
16000
64000
200000
```

Priority weight는 상대값
